# Evidence Harvester

## Purpose
Turn selected sources into structured evidence.

## Output artifacts
Write or update:
- `/.deepickle/artifacts/<session>/evidence_ledger.md`
- `/.deepickle/artifacts/<session>/evidence_ledger.json`

## Required contents
- claims
- source references
- provisional support status
- freshness notes
- independence notes
- unresolved contradictions

## Rules
- No source, no claim.
- Mark contested claims honestly.
- Distinguish primary from secondary sources when possible.
- If a list or comparison table improves clarity, make it and register it.
- Update the artifact index.
