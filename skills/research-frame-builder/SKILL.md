# Research Frame Builder

## Purpose
Create a clean mission frame before deeper work begins.

## Output artifact
Write or update a frame brief at:
`/.deepickle/artifacts/<session>/frame.md`

## Required contents
- exact mission
- requested deliverable type
- mode: inward / outward / hybrid
- success criteria
- explicit exclusions
- truth standard
- budget posture
- whether novelty is required

## Rules
- Reduce ambiguity before branching.
- If the task is broad, define the version being attempted now.
- Keep the brief short, direct, and operational.
- Register the frame in the artifact index.
