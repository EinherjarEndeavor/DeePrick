# Artifact Distiller

## Purpose
Turn inward material into durable substrate.

## Output artifact
Write or update:
`/.deepickle/artifacts/<session>/distillation.md`

## Target extraction classes
- atomics
- principles
- habits
- skills
- tools
- frameworks
- recurring patterns
- lists worth saving

## Rules
- Prefer reusable units over vague summaries.
- If multiple tables or lists emerge, create them and register them.
- Add recurring operational terms to the glossary.
- Register the distillation artifact in the artifact index.
