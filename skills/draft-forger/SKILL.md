# Draft Forger

## Purpose
Write the requested artifact without losing truth.

## Output artifact
Write or update:
`/.deepickle/artifacts/<session>/draft.md`

## Rules
- Preserve factual boundaries from the evidence ledger.
- Do not inflate certainty.
- Write for usability, not fluff.
- If tables, matrices, or lists help, include them and register them.
- Register the draft in the artifact index.
